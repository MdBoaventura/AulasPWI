Tabela

funcionalidade das tags html

Tag Função
h1,h2,h3... Titúlo
Strong Negrito
p Começa um paragrafo e quebra linha
br / quebra a linha
b Negrito na versão html 4
i Itálico
hr / cria uma linha
ul cria uma lista
ol cria uma lista ordenada
charset="1" reconhece e organiza a acentuação
a cria um link
title define o titulo, fica no cabeçalho
style permite incluir elementos de estilo na tag(css)
!-- cria um comentário não visivel
head envolve o cabeçalho
body envolve o corpo, com textos e tags
<> abre e facha a tag
li cria um item na tabela
menu cria uma lista de itens
em maior ênfase em itálico
var um espaço a serreservado por uma variavel
cite adiciona uma citação

Atributos

Id: São números ou letras que identificam algo, esses números e letras não podem ser repitidas.

Style e Class: dentro desses atributos você insere tags que vão estilizar seu site
